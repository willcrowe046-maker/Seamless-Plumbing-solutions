# Seamless Plumbing Solutions — Website

Static HTML/CSS/JS website for Seamless Plumbing Solutions, Ermington NSW.

## Structure

```
├── index.html        Home page
├── services.html     Services + price estimator
├── emergency.html    24/7 emergency page
├── about.html        About & team
├── contact.html      Booking form & contact details
├── css/styles.css    Full design system
└── js/main.js        Interactive behaviour
```
